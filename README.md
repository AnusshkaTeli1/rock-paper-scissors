# 🎮 Rock Paper Scissors Game

An interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript.

## Features
- 3-second animated countdown timer before every round
- Circular ring timer that turns red when time is almost up
- Auto-selects a random move if you don't pick in time
- First to 10 wins takes the match — game ends automatically
- Animated game over screen with confetti if you win
- Live scoreboard with progress bars tracking both players
- Score pulse animation when a point is scored
- Fully responsive and runs in any browser — no installation needed

## How to Play
1. Open `rock_paper_scissors.html` in your browser
2. Click START to begin a round
3. Pick 🪨 Rock, 📄 Paper, or ✂️ Scissors before the timer runs out
4. First player to win 10 rounds wins the match!

## Rules
- Rock crushes Scissors
- Scissors cuts Paper
- Paper covers Rock

## Technologies Used
- HTML5
- CSS3 (animations, transitions, flexbox)
- Vanilla JavaScript (game logic, countdown timer, DOM manipulation)
